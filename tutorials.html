Document Scanner

Use Scanner when you want a clean digital copy of a paper document before OCR, review, or sharing.

How To Use It

  1. Open the ProdX Keyboard main workspace.
  2. Choose Scanner from the task list.
  3. Capture a document with the camera, or import an existing photo from gallery/storage.
  4. Use the scanner screen to crop, rotate, clean, and adjust pages.
  5. Press the scanner finish/next button.
  6. Choose where to save the cleaned PDF or JPEG using Android's save picker.
Scanner is for cleaning and saving document images. It is not a general PDF editor.

Best For

  • Paper forms, printed letters, receipts, notes, and book pages.
  • Photos that need crop/rotation cleanup before OCR.
  • Multi-page paper scans that should be saved as a clean PDF.

Scanner To OCR

Scanner and OCR are separate tools. Scanner prepares the page. OCR extracts editable text.

Recommended Flow

  1. Run Scanner first to clean and save the paper document.
  2. Return to the main workspace.
  3. Choose OCR text extract.
  4. Attach the cleaned scan you saved.
  5. Run OCR to extract editable text.
Keeping scanning separate from OCR means you keep a clean visual copy even if you later need to re-run OCR or review the original page.

Which Tool Should I Use?

  • Use Scanner when the image needs crop, rotate, cleanup, or a saved PDF/JPEG.
  • Use OCR text extract when you already have a readable image/PDF and need text.
  • Use both when the source is a paper document or a messy photo.

Cleanup Warning

Some AI, OCR, document, and sharing workflows can leave result text in the clipboard or temporary AI work files. ProdX shows a cleanup warning when there is something real to clean.

What The Buttons Mean

  • Clear clipboard removes ProdX-owned clipboard content.
  • Delete temp files removes temporary AI work files created for processing.
  • Later closes the warning without cleaning everything immediately.

Why The Warning May Stay Open

The warning closes only after both cleanup areas are clear, or when you press Later. For example, if you clear the clipboard but temporary AI files still exist, the warning remains so you can finish cleanup in one place.

On resume, ProdX checks for actual cleanup residue first. The warning should not appear just for the sake of appearing.
